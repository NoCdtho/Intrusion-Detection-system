## Intrusion Detection System ##
OVERVIEW:
IDS is the repository for the mini-project I built during my 6th sem where
I learnt and implemented RL model to detect network attacks on IOT device.
Furthermore, I have implemented LSTM on my RL model to learn sequential attack patterns.
We used NSL-KDD dataset but it was old and contains simulated attack patterns so we moved to Beth-dataset 
to work on real world attack patterns.
